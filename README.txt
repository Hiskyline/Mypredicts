MONARCH AI SERVER

How to run locally:

1. Install Node.js
2. Open terminal in this folder
3. Run:

npm install
npm start

Server will run at:
http://localhost:3000

API endpoints:

GET /api/ext/predictions
POST /api/ext/game-result

To deploy online:

1. Upload this project to GitHub
2. Deploy using Render.com or Railway.app
3. Use the deployed URL as BASE_URL in your userscript
